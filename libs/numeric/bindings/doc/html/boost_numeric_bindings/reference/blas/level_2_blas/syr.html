syr
Prototype

There is one prototype of syr available, please see below. 

syr( const Scalar >, const VectorX& x, MatrixA& a );

Description

syr (short for symmetric rank-1 update) provides a C++ interface to BLAS routines SSYR and DSYR. syr performs the symmetric rank 1 operation

A := alpha*x*x' + A,

where alpha is a real scalar, x is an n element vector and A is an n by n symmetric matrix.

The selection of the BLAS routine is done during compile-time, and is determined by the type of values contained in type VectorX. The type of values is obtained through the value_type meta-function typename value_type<VectorX>::type. Table X below illustrates to which specific routine this dispatching will take place.
