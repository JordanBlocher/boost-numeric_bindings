tftri
Prototype

There is one prototype of tftri available, please see below. 

tftri( MatrixA& a );

Description

tftri (short for $FRIENDLY_NAME) provides a C++ interface to LAPACK routines STFTRI, DTFTRI, CTFTRI, and ZTFTRI. tftri computes the inverse of a triangular matrix A stored in RFP format.

This is a Level 3 BLAS version of the algorithm.

The selection of the LAPACK routine is done during compile-time, and is determined by the type of values contained in type MatrixA. The type of values is obtained through the value_type meta-function typename value_type<MatrixA>::type.
