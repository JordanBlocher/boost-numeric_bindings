gelqf

Prototype

There is one prototype of gelqf available, please see below. 

gelqf( MatrixA& a, VectorTAU& tau );

Description

gelqf (short for $FRIENDLY_NAME) provides a C++ interface to LAPACK routines SGELQF, DGELQF, CGELQF, and ZGELQF. gelqf computes an LQ factorization of a complex M-by-N matrix A: A = L * Q.

The selection of the LAPACK routine is done during compile-time, and is determined by the type of values contained in type MatrixA. The type of values is obtained through the value_type meta-function typename value_type<MatrixA>::type.
