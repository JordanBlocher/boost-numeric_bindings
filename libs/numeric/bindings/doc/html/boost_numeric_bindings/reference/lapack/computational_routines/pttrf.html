pttrf
Prototype

There is one prototype of pttrf available, please see below.

Description

pttrf (short for $FRIENDLY_NAME) provides a C++ interface to LAPACK routines SPTTRF, DPTTRF, CPTTRF, and ZPTTRF. pttrf computes the L*D*L' factorization of a complex Hermitian positive definite tridiagonal matrix A. The factorization may also be regarded as having the form A = U'*D*U.

The selection of the LAPACK routine is done during compile-time, and is determined by the type of values contained in type VectorD. The type of values is obtained through the value_type meta-function typename value_type<VectorD>::type.
