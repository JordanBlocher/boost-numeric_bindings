ppsv

Prototype

There is one prototype of ppsv available, please see below. 

ppsv( MatrixAP& ap, MatrixB& b );

Description

ppsv (short for $FRIENDLY_NAME) provides a C++ interface to LAPACK routines SPPSV, DPPSV, CPPSV, and ZPPSV. ppsv computes the solution to a complex system of linear equations A * X = B, where A is an N-by-N Hermitian positive definite matrix stored in packed format and X and B are N-by-NRHS matrices.

The Cholesky decomposition is used to factor A as A = U*H U, if UPLO = 'U', or A = L * L**H, if UPLO = 'L', where U is an upper triangular matrix and L is a lower triangular matrix. The factored form of A is then used to solve the system of equations A * X = B.

The selection of the LAPACK routine is done during compile-time, and is determined by the type of values contained in type MatrixAP. The type of values is obtained through the value_type meta-function typename value_type<MatrixAP>::type.
