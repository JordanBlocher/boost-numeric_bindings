geqrf
Prototype

There is one prototype of geqrf available, please see below. 

geqrf( MatrixA& a, VectorTAU& tau );

Description

geqrf (short for $FRIENDLY_NAME) provides a C++ interface to LAPACK routines SGEQRF, DGEQRF, CGEQRF, and ZGEQRF. geqrf computes a QR factorization of a complex M-by-N matrix A: A = Q * R.

The selection of the LAPACK routine is done during compile-time, and is determined by the type of values contained in type MatrixA. The type of values is obtained through the value_type meta-function typename value_type<MatrixA>::type.
