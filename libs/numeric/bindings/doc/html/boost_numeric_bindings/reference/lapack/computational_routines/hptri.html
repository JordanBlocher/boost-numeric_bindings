hptri
Prototype

There is one prototype of hptri available, please see below. 

hptri( MatrixAP& ap, const VectorIPIV& ipiv );

Description

hptri (short for $FRIENDLY_NAME) provides a C++ interface to LAPACK routines SSPTRI, DSPTRI, CHPTRI, and ZHPTRI. hptri computes the inverse of a complex Hermitian indefinite matrix A in packed storage using the factorization A = UD*U*H or A = LD*L*H computed by ZHPTRF.

The selection of the LAPACK routine is done during compile-time, and is determined by the type of values contained in type MatrixAP. The type of values is obtained through the value_type meta-function typename value_type<MatrixAP>::type.
